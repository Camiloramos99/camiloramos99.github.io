BEST BUY MARKET

E-commerce escalable desarrollado con React, Vite y Tailwind CSS, desplegado en Netlify

E-commerce desarrollado desde cero con navegación intuitiva, filtros avanzados, carrito dinámico y autenticación de usuarios. Su diseño está cuidado en cada detalle, con un flujo pensado para ofrecer una experiencia ágil y clara. Este proyecto me permitió profundizar en la construcción de aplicaciones escalables y en la integración de APIs para la gestión de productos.

Integración con API propia para la gestión de productos

Desarrollé una API personalizada con 20 productos alojada en Render, que la app consume mediante fetch. Para evitar demoras en la primera carga, implementé una solución UX: mientras se espera la respuesta, se renderiza un JSON local con los mismos productos, asegurando una experiencia fluida desde el inicio. Una vez que la API responde, los datos se actualizan dinámicamente.

Sistema de registro de órdenes

Cada compra se guarda con fecha, total y cantidad de productos mediante localStorage, permitiendo al usuario consultar su historial desde “My Orders”. Cada entrada es interactiva y puede expandirse para ver los detalles completos. Esta funcionalidad me permitió practicar el manejo de estado, la persistencia local y la organización de datos dinámicos orientados a la experiencia del usuario.

Autenticación de Usuarios

La aplicación cuenta con un sistema de autenticación que permite a cada usuario registrarse, iniciar sesión y acceder a secciones personalizadas como “My Orders y “My Account”. Implementé esta funcionalidad con validación de formularios, control de rutas protegidas y manejo de estado global. Este desarrollo me permitió profundizar en flujos de login seguros, gestión de sesiones y experiencia personalizada.

Carrito de Compras

El carrito permite agregar, editar y eliminar productos en tiempo real, reflejando los cambios de forma inmediata en la interfaz. Implementé esta funcionalidad con manejo de estado reactivo y lógica condicional, asegurando una experiencia fluida y controlada. Este desarrollo me ayudó a profundizar en la gestión de componentes dinámicos, actualizaciones locales y diseño centrado en la interacción del usuario.

Filtros Avanzados

La aplicación permite buscar productos mediante una barra de búsqueda dinámica y filtrar por categorías específicas, mejorando la navegación y la relevancia de los resultados. Implementé esta funcionalidad con lógica condicional y renderizado reactivo. Me permitió profundizar en el manejo de eventos, estados controlados y diseño centrado en la experiencia del usuario.

Vista detallada de cada producto con información completa

Diseñé una interfaz limpia y enfocada para mostrar información clave de cada producto: imagen destacada, nombre, precio y descripción. Esta vista se presenta en un modal con diseño minimalista, priorizando la jerarquía visual y la claridad. Me permitió practicar composición visual, manejo de props dinámicos y diseño orientado a conversión.

Footer personalizado con branding y accesos clave

El proyecto incluye un footer diseñado para reforzar la identidad visual de Best Buy Market, con enlaces útiles para el usuario y presencia en redes sociales. Esta sección cierra la experiencia con coherencia estética, accesibilidad y profesionalismo, integrando navegación secundaria, íconos interactivos y créditos personalizados.
