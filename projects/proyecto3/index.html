TOMÁS FERREYRA PH

Landing page fotográfica hecha con HTML, CSS y Bootstrap desplegado en Netlify.

Landing page moderna y totalmente responsive , pensada para mostrar cómo un fotógrafo profesional puede presentar su trabajo en la web de manera atractiva y funcional.
Creada como demo para mi portfolio, el objetivo fue destacar el trabajo visual, integrar elementos de identidad personal y ofrecer una navegación fluida en cualquier dispositivo. Además, cuidé detalles de accesibilidad, rendimiento y optimización de imágenes (WebP), logrando una página ligera y escalable.

Responsive Design

La página se adapta a distintos dispositivos, con layouts pensados tanto para desktop como mobile.
Mejoré la implementación de Bootstrap y CSS responsive, entendiendo cómo ajustar componentes y media queries para garantizar accesibilidad y consistencia visual en todos los tamaños de pantalla.

Home (Hero)

Encabezado con el nombre e identidad del fotógrafo, acompañado de un diseño limpio y profesional. Incluye un botón CTA “View My Work” que guía al usuario hacia la galería, además de íconos de redes sociales en color blanco, discretos pero visibles, que complementan la sección sin competir con la llamada principal a la acción.
Aprendí a crear un hero visualmente atractivo y con jerarquía clara, reforzando la importancia de la primera impresión y el correcto balance entre elementos de branding, redes sociales y acciones clave para el usuario.

Gallery

Galería de fotos optimizada con dos experiencias: efecto acordeón horizontal en desktop (implementado únicamente con CSS) y visualización táctil tipo “pasa hojas” en mobile.
Este reto me permitió sacar el máximo provecho de CSS para interactividad, entendiendo cómo usar transiciones y estados para lograr un resultado dinámico sin JavaScript. Además, profundicé en la adaptación de la experiencia según el dispositivo y la optimización de imágenes en WebP para mejorar el rendimiento.

About Me

Sección con foto y breve descripción que transmite identidad y cercanía.
Aprendí a estructurar secciones de marca personal usando HTML semántico y a resaltar información clave de forma simple y atractiva.

Contact Me

Sección con un formulario de contacto de diseño moderno, pensado para ser intuitivo y fácil de completar.
Trabajé en la estructura visual y accesibilidad del formulario, entendiendo cómo el diseño influye en la experiencia del usuario y cómo dejarlo listo para futuras integraciones de funcionalidad.
Adicionalmente tiene un pie de página sencillo para concluir este proyecto.
